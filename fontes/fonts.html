Como podemos estar modificando as fontes no CSS?

Font Family

declaramos uma familia de fonte para o elemento que estamos selecionando


        p {
            font-family: 'Courier New', Courier, serif;
        }

Esse atributo possui Fallback, caso não encontre a fonte principal selecionada, estara procurando a segunda declarada.

Font Weigth

O font-weigth como o proprio nome ja diz retrata o peso da fonte, se ela é bold, light, bolder ou até mesmo niveis numericos. como 100, 200, 300.


        p {
            font-weight: bolder;
        }

Font Style

Declara o estilo da fonte, como por exemplo o italic e oblique.


        p {
            font-style: italic;
        }

Font Size

Definimos o tamanho da fonte.


        p {
            font-size: 100px;
        }

Outras modificações com fontes.

Letter spacing e word spacing

Letter-spacing representa o espaço entre as letras dentro de um paragrafo, ja o word-spacing representa o espaçamento entre as palavras, ambas devem ser declaradas com uma unidade de medida.

Line-height e text-transform

Line-height representa o espaço entre as linhas, text-transform podemos transformar o texto, como por exemplo, deixar todo o texto em letra maiúscula com o valor uppercase, minuscula com lowercase, e entre outras transformações.

text-decoration

demonstra a decoração do texto como por exemplo overline, underline, line-throught

text-align

Essa propriedade representa o alinhamento do texto no paragrafo, como center, left, right ou justify

text-shadow

Adicionamos sombra no texto.


        /* deslocamento-x | deslocamento-y | raio-de-desfoque | cor */
        text-shadow: 1px 1px 2px black
