Quais são e como funciona o posicionamento dos elementos inblock no html?

As propriedades podem ser alteradas no CSS.

o nome do atributo que utilizamos é position, com ele podemos utilizar os valores, relative que não altera o fluxo em que as divs estão posicionadas, absolute na qual o fludo das demais divs são alteradas, ja o valor fixed, como o proprio nome ja diz é utilizado para deixar o elemento fixo na tela, como por exemplo em navsBar em que acompanham sempre fixas na tela.

Podemos tambem trocar a ordem de empilhamento das div utilizando a propriedade z-index.

Trabalhando com flex-direction

Para trabalharmos com flex-direction devemos declarar na div pai o display:flex;com ele tornamos a div flexivel, e podemos devolve-las em formato de coluna através do flex-direction:column;

Red
Green
Blue


        .container {
            display: flex; /* Dessa forma deixamos as caixas uma do lado da outra */
            flex-direction: column; /* Agora deixamos em forma de coluna */
            justify-content: center; /* Colocamos o elemento no centro */
        }

Trabalhando com grid

O grid pode ser considerado uma forma alternativa do Flexbox, com ela podemos dividir a tela em areas e dessa forma informar quais divs devem estar em cada area e dessa forma tambem podemos estar indicando quanto cada area deve ter de tamanho ou largura.

red
green
blue
yellow


        .grid {
            display: grid;
            grid-template-areas: 
            "header header"
            "main aside"
            "footer footer";
            grid-template-rows: 30px 400px 40px;
            grid-template-columns: 400px 100px;
        }
        
        .red2 {
            grid-area: header;
            background-color: red;
           }
           
        .green2 {
            grid-area: main;
               background-color: green;
           }
           
        .blue2 {
            grid-area: aside;
            background-color: blue;
           }
        
        .yellow {
            grid-area: footer;
            background-color: yellow;
        }
