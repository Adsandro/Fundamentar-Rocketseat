O que são box model?

De uma forma leiga de se falar, são apenas retangulos que possuem propriedades, podendo conter caixas dentro de caixas.

Algumas de suas propriedades

Tamanho (altura e largura) : width e height

Conteudo: content

Bordas: Border

Preenchimento interno: peeding

Espaçamento fora da caixa: margin

Box Sizing

Box Sizing representa o tamanho total de uma box no css, no exemplo a seguir o box sizing possui 140px, por possuir uma largura de 100px e padding de 20px em cada lado.


        div {
            width: 100px;
            height: 100px;
            margin: 10%;
            padding: 0 20px;
        }

Caso queira que a box possua os 100px solicitado em width deve-se implementar o box sizing: border-box;

Display-block-inline

de forma simple, esse display mostra como os elementos serão representados dentro da <div>, seja em bloco ou em linhas.

Block: <p> <div> <selection> <h1>

Inline: <strong> <b> <span> <em>

Margem (margin)

Esse elemento representa o espaçamento entre os objetos presentes no HTML.

- margin-top | margin-right | margin-bottom | margin-left



        div {
                margin: 12px 16px 10px 4px;
                margin: 12px 16px 0;
                margin: 8px 16px;
                margin: 8px;
            }

O elemento margin possuem o valor auto que ajuda de forma automatica a largura da margem, mantendo a <div> no centro e o top e bottom em 0.

Cuidado com margin collapsing (top se ajunta ao bottom)

Padding

Representa o espaçamento interno dos elementos do HTML, e possuem as mesmas propriedades do Margin.


        div {
            padding: 12px 16px 10px dpx;
            padding: 12px 16px 0;
            padding: 8px 16px;
            padding: 8px;
            }

Bordas (border)

Valores : “<border-style>" | “<border-width> | ~<border-color>"

Para que o elemento funcione é essencial que seja declarado todos os valores anteriores.
