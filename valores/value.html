Unidades de medidas utilizadas frequentemente

<length> Representa um valor de distância:px,em,VW

<angle> Representa um ângulo:deg,rad,turn

<time> Representa um tempo:s,ms

<resolution> Representa resoluções para dispositivos:dpi

Distancias relativas

As distancias relativas são excelentes por conta da adaptatividade com a tela. diferente das distancias absolutas que não possuem uma responsividade predefinida. como por exemplo o px elemento mais utilizado

em :Tamanho da font do elemento pai

rem :Tamanho da font do elemento raiz (root/html)

vw :1% da viewport wid

vh :1% da viewport height

O elemento raiz pode ser trocado da seguinte forma:

 :root {
        font-size: 18px;
    }
    
    /* OU */
    
    html {
        font-size: 18px;
    }
