O que são At rules?

At rules como o próprio nome ja diz, refere-se a regras de arrouba (@), segue alguns exemplos.

@import

Utilizado especialmente para adicionar links externos no nosso HTML, como por exemplo, links de fontes.

@media

Utilizado para passar informações do tamanho do dispositivo.

 @media(min-width:500px){
        "Deve ser informado as regras nesse local"
     }

@font-face

Utilizado para adicionar apenas fontes externas.

@keyframes

Especifico para animações com o CSS.
