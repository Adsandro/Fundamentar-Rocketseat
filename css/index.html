Como adicionar o CSS no HTML?

Pelo elemento Style

Existem diversas formas de adicionar o CSS ao Html, uma delas pelo elemento style="" dentro das tags.

Link CSS no head

A forma mais comum utilizada é pelo head, ou cabeçalho no arquivo HTML, ele pode ser representado da seguinte forma.

<link rel="stylesheet" href="style.css">

Desa forma, devemos informar em href="" o local do arquivo css que utilizaremos

Style no head

Essa forma consiste em utilizar a tag Style dentro do head do HTML. 

            
                <style>
                    h1 {
                        color: blue;
                        font-size: 40;
                        background-color: red;
                        width: 300px;
                    }
                </style>

Pelo @import

O @import é utilizado dentro da pagina de CSS, em especifico ele possui a função de importar informações da Web, como por exemplo fonts que estão cadastradas no google fonts, segue exemplo de como utilizar.

@import url('https://fonts.googleapis.com/css2?family=Oleo+Script+Swash+Caps&display=swap')

Porem recomenda-se que seja utilizada em formato de link dentro do head do HTML da seguinte forma. 


        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
        <link href="https://fonts.googleapis.com/css2?family=Oleo+Script+Swash+Caps&display=swap" rel="stylesheet">
