Como funciona o background?

Propriedades de um background

Background-color: red; Esse elemento troca a cor do fundo do elemento selecionado para a cor desejada.

background-image: url("link") Assim como o color, pode ser colocado uma imagem. Esse elemento pode conter outras instruções adicionais como background-repeat: no-repeat; Dessa forma a imagem não ira se repetir caso ela não preencha toda a box. Ou podemos utilizar o repeat-y ou repeat-x para informar em qual eixo ela pode se repetir.

background-position: right; Com esse elemento podemos selecionar em qual posição a imagem do background deve ficar.

background-size:cover; Background-size pode ser utilizado para informar o tamanho dele, podemos utilizar unidades de medidas, porcentagem, ou o termo cover que estica a imagem para se encaixar completamente no local, ou o termo contain fazendo com que a imagem se encaixe perfeitamente no local.

background-origin; e background-clip; Ambos são utilizados para informar como a imagem do BG deve se encaixar.

background-attachment: fixed; O valor padrão para esse elemento é scroll fazendo com que a imagem do background siga os elementos da frente ao rolarmos para baixo, ja o valor fixed pode ser utilizado para deixar a imagem de como fixo, sem mudar.

background: linear-gradient(red, yellow); Com esse elemento o background gera um gradiente na qual vai de uma cor para outra, podemos tambem informar a quantidade de graus na qual queremos que seja feito.
