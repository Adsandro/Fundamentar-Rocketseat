Tipos de cores.

  • Background-color
  • Color
  • Border-color

Valores

  • Palavra-chave (Blue, Green, red)
  • Hexadecimail (#0000)
  • Funções: rgb, rgba, hsl, hsla

Formas de utilizar

Por currentcolor

color: currentcolor; dessa forma o elemento selecionado herdara a referencia de cor do elemento pai.

color: red; Nesse modelo estaremos informando diretamente qual cor deve ser colocada na div filho.

Hexadecimail


        color: #090;
        color: #009900;
        color: #090a;
        color: #009900a;

As duas primeiras formas representam apenas o RGB, ja as duas ultimas representam o RGBA com a inclusão do alpha que da a transparencia para a cor.

RGB

O RGB possui uma variação de 0 a 255 em sua paleta de cores.


        color: rgb(34, 12, 64, 0.6);
        color: rgba(34, 12, 64, 0.6);
        color: rgb(34 12 64 / 0.6);
        color: rgb(34 12, 64 / 0.3);
        color: rgb(34.0 12, 64 / 60%);
        color: rgb(34.0 12, 64 / 30%);

HSL

Pode ser considerado um elemento mais complexo para cores, possuindo a seguinte sintaxe:


        color: hsl(30, 100%, 50%, 0.6); 
        color: hsla(30, 100%, 50%, 0.6);
        color: hs1(30 100% 50% / 0.6);
        color: hsla(30 100% 50% / 0.6);
        color: hs1(30.0 100% 50% / 60%); 
        color: hsla(30.2 100% 50% / 60%);
