Meta dados para redes sociais

Os meta dados para redes sociais servem para tornar o compartilhamento de um link mais agradavel quando for compartilhado, dessa forma podendo gerar uma imagem de compartilhamento, gerar um texto e até mesmo outras informações. Cada rede social utiliza seu elemento Meta especifico
